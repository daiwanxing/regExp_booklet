### 正则模式速查

| 模式 | 说明 |
| ------ | ------ |
| [a-z0-9] | 匹配a-z 0-9之间的任意字符 |
| [abc] | 匹配abc之间的任意字符 |
| [^abc] | 匹配非abc之间的任意字符 |
| \&[^;+]; | 匹配&符号拼接非";"的任意字符一次或多次，并以“;”结尾 |
| \d | 匹配数字 | 
| \d{1,} | 匹配数字，至少匹配一次, 至多贪婪匹配全部连续数字 | 
| \d{1,10} | 匹配数字，至少匹配一次, 至多贪婪匹配十次连续数字|
| \d{1,10}? | 匹配数字，惰性匹配, 只完成一次匹配（太懒了，完成任务就行）| 
| \w | 匹配单词(字母)|
| \W | 匹配非单词|
| \s | 匹配空白字符|
| \S | 匹配非空白字符|
